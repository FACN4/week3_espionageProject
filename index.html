Glad you made it!

Welcome to the official FAC campus comparison site. This a site for members of Founders and Coders (FAC) to checkout what all the other campuses are up to. You could call it spying, but we prefer not to. There are two main sections.

The Stats:
This looks at the repositorys created in the last 7 days and shows the total commits and per capita commits made by each Campus. This data is live and fresh API calls are made when the page is refreshed. In time, this page could include more detailed analysis way the various campuses use Github. This could inclue 'famous last words' (the final commit message of a project), name and shame branch names or commits with stupid names like 'whateva', 'Updates' or 'blah'. The sky's the limit.

Which Campis is Which:
Give a small bio on each of the Campuses and and a location relevent photo. Future additions could be total number of students, the current mentors, team photos and much much more.
